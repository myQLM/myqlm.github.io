qat.synthopline.interface.OperatorTableauÔÉÅ

class qat.synthopline.interface.OperatorTableauÔÉÅ

A class representing a Clifford Tableau.

A detailed introduction can be found in [dB12].

Usage example:

In this example, we use the Tableau structure to conjugate a Pauli operator \(ZZ\) with the Clifford operator that can be implemented via a H gate on qbit 1 and a CNOT gate from 0 to 1.

The resulting Pauli operator is \(IX\) and no phase is picked up.

from qat.synthopline.interface import OperatorTableau, WeylOperator

tableau = OperatorTableau(2)

tableau.h_right(1)
tableau.cnot_right(0, 1)

operator = WeylOperator("ZZ")

operator.conjugate_with(tableau)

phase, pauli = operator.to_pauli()
print(phase, pauli)
