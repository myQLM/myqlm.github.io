qat.synthopline.phase_polynomials.random_phase_polynomialÔÉÅ

qat.synthopline.phase_polynomials.random_phase_polynomial(nbqbits, nbterms, no_trivial=True)ÔÉÅ

Produces a random phase polynomial over nbqbits qbits and with support of size nbterms.

Note

The seed of the generator can be controlled using numpy‚Äôs random.seed method.

import numpy as np
from qat.synthopline.phase_polynomials import random_phase_polynomial

np.random.seed(1234)
ppol = random_phase_polynomial(10, 15)
for key, value in ppol.items():
    print(key, ":", value)
