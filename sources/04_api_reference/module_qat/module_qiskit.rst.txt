.. module:: qat.qiskit


qat.qiskit
===========

:mod:`qat.qiskit` is a module that allows to use the qaptiva backend with the qiskit 2.0 API. With it is possible
to run Samplers and Estimators that can use a qaptiva backend and return qiskit data stuctures.

.. autoapi::

        [:class:`qat.qiskit.Sampler`] A high level qiskit Sampler that wraps qaptiva QPUs
        [:class:`qat.qiskit.Estimator`] A high level qiskit Estimator that wraps qaptiva QPUs
        [:class:`qat.qiskit.QaptivaService`] Instantiate a qiskit backend and wraps it around a remote QPU

Some relevant functions:

.. autoapi::

        [:func:`qat.qiskit.converters.qiskit_to_qlm`] An internal function that converts qiskit circuits to qlm circuits
