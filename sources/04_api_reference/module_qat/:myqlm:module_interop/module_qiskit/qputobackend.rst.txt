qat.interop.qiskit.QPUToBackend
===============================

.. class:: qat.interop.qiskit.QPUToBackend(*args: Any, **kwargs: Any)

   Basic connector implementing a Qiskit Backend, pluggable on a QLM QPU.

   :param qpu: :class:`~qat.qpus.QPUHandler` object.
   :param configuration: BackendConfiguration object. Leave default value for
                         standard uses.
   :param provider: Provider responsible for this backend.

   .. method:: set_qpu(qpu)

      Sets the QLM QPU that this backend is supposed to use.

      :param qpu: QLM QPU object.
